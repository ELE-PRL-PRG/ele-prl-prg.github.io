Evolving Language Ecosystems (ELE) Evolving Language Ecosystems (ELE)

A computer language is more than the code that the programmers write or the compiler used to translate it into machine instructions. Modern languages are characterized by a rich ecosystem that includes a multitude of source of information from open source code repositories, help pages and manuals, all the way to discussion forums such as Stackoverflow.

The ELE project explores the fundamental techniques and algorithms for evolving entire language ecosystems. Our purpose is to reduce the cost of wide-ranging changes to programming languages and obviate the need for devising entirely new languages.
